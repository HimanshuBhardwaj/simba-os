Getting started.

This page shows you how to set up Simba for your project.

You are here

As the picture shows ....

This you need to configure

Look at the reference implementation and find the config file. In there you need to adapt following properties:


		simba.url=http://localhost:xxx/the-location-of-xxx
		simba.web.url=http://localhost:xxx/the-location-of-xxx

After this you can configure your own chain with the commands. See the more details page for a list of all the commands.
