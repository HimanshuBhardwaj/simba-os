Want to contribute?

If you decide you want to add features to Simba or improve something feel free to do so. Just get the source from github and start changing! Following steps will show you where you can adapt vital parts.

Project structure

Here's is the what-is-what of our multi-module maven project:
  • simba: the parent pom
  • simba-ri: the reference implementation
  • simba-manager: the AngularJS gui to mananage users & roles.
  • simba-webdriver-test: webdriver/selenium tests to detect regression.

The projects of most importance to you would be simba-ri and simba-zoo to start with.
